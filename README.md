# Customer Behavior Analysis

This project analyzes customer purchasing behavior using Python, MySQL and Power BI.

## Technologies Used
- Python
- Pandas
- NumPy
- MySQL
- Power BI

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. SQL Queries for Data Extraction
5. Power BI Dashboard Visualization

## Files Included
- customer_behavior.ipynb – Data analysis notebook
- dataset.csv – Dataset used
- mysql_queries.sql – SQL queries used
- dashboard.pbix – Power BI dashboard

## Insights
- Customer purchase patterns
- Revenue distribution
- Category-wise sales analysis
- Customer segmentation
